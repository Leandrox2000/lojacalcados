Sobre Nós



Somos um grupo formado por três pessoas, objetivando a criação de uma loja de calçados.

O projeto otimiza a aplicação de todas as etapas teóricas adquiridas durante o curso de forma integrada, visando uma melhor experiência dos clientes, através de requisitos funcionais como:

Requisitos funcionais:


  • Cadastro e Login de usuário
  • Barra de comentários
  • Acesso a histórico de compras e transações
  • Efetuar compras
  • Realizar pesquisas
  • Consultar dados dos clientes
  • Solicitar um atendimento

    • Requisitos não-funcionais:


  • Segurança
  • Confiabilidade
  • Facilidade de uso
  • Desempenho
  • Disponibilidade

    • Na disciplina do Projeto Integrador III, o objetivo será dar continuidade no desenvolvimento de software, com foco na implementação e melhorias para uma loja virtual.
    Serão utilizadas seguintes tecnologias e ferramentas:


  • IntelliJ IDEA Community Edition
  • Visual Studio Code
  • GitHub
  • Java 11 com Spring Boot
  • Angular CLI
  • MySQL
  • NodeJS e Tomcat 9
